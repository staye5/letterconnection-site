# Letter Connection Website

Static first-pass website for `letterconnection.com`.

## Files

- `index.html` - full landing page
- `styles.css` - responsive styling
- `script.js` - small header behavior
- `assets/` - brand concept imagery derived from the approved design board

## Current Positioning

The page should present Letter Connection as a clean, credible mailbox and document-services brand. Do not use current renovation or unfinished store photos as public-facing hero imagery. Real finished-store photos can replace the concept images after the repaint/remodel looks ready.

Primary website job:

1. Make mailbox rentals and business addresses the clearest offer.
2. Convert visitors into calls, directions, mailbox starts, renewals, or print-file emails.
3. Present shipping, print/copy/scan, fax, notary, and package support without overpromising services still being stabilized.
4. Compete locally against The UPS Store, FedEx Office, Regus, PostNet, PostalAnnex, and independent mail centers by being clearer, more local, and easier to act on.
